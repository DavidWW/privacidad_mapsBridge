Política de privacidad

David Margalef, propietario de la aplicación, en adelante, "MapsBridge", pone en conocimiento de los usuarios la política llevada a cabo respecto al tratamiento de todos los datos que por funcionalidades del software se faciliten dentro de la aplicación.

Recogida y tratamiento de datos de carácter personal

MapsBridge no recoge ningún dato de carácter personal.

En caso autenticación, se realizará a través del API pública de Google. En ese caso, únicamente se lleva a cabo este proceso para recuperar los datos que el usuario generó en los mapas de Google. MapsBridge no conocerá en ningún momento la contraseña de acceso. El proceso de autenticación se realizar mediante el propio sitio web de Google, imposibilitando esta acción.

Una vez se realice el proceso de autenticación, los únicos datos recuperados son los seleccionados por el cliente (sus mapas).

No se recuperará ni almacenará el email del usuario.

En ningún caso la información obtenida del usuario guardará ningún tipo de relación con información personal privada o sensible.

Propósito del tratamiento de datos de carácter personal

MapsBridge no recoge datos de carácter personal.

Recopilación y tratamiento de datos de carácter no personal

No se recogen datos.

Divulgación a terceros

MapsBridge no recoge datos de carácter personal, por lo tanto no existen datos de este tipo a compartir con terceros.

Protección de datos de carácter personal

MapsBridge no recoge datos de carácter personal.

Servicios de localización

MapsBridge no accede en ningún momento a servicios de localización o posicionamiento.
